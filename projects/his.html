Homographic Image Stitching

Project Overview

This project explores the use of homographic transformations for image stitching, where multiple overlapping photographs are seamlessly merged to form a single panoramic image. This technique finds extensive use in photography, virtual reality, and geographic imaging.

Homographic transformations account for the changes in perspective, allowing for alignment of images taken from different angles. The accuracy of this process is critical, particularly in achieving loop closure, where the start and end of the panoramic sequence meet.

Technical Implementation

The implementation leverages several computer vision techniques to enhance the quality and precision of the stitching:

  • Feature Detection and Matching: Key points are detected in each image using the SIFT algorithm, which are robust to changes in scale, rotation, and illumination. Corresponding points between overlapping images are matched based on feature similarity.
  • Estimation of Homography: For each pair of matched images, a homographic matrix is calculated that best aligns them into a common plane.
  • Image Warping and Blending: Images are warped (transformed) according to calculated homographies, and then blended smoothly to avoid visible seams.
  • Optimization with GTSAM: Global optimization techniques are applied using the GTSAM library to refine all homography matrices simultaneously, ensuring global consistency and improving loop closure.

Results and Discussion

The results demonstrate significant improvements in image alignment and panorama smoothness after optimization. The loop closure, critical for the panoramic integrity, is specifically enhanced, providing a virtually seamless image even at the joining edges.
