Lidar-Camera Sensor Fusion with Semantic Segmentation

  • GitHub URL: Project Link

This project demonstrates the integration of Lidar and camera data using advanced computer vision techniques to enhance autonomous navigation systems. It includes semantic segmentation to classify different environmental elements, which is crucial for path planning and obstacle avoidance in autonomous vehicles.

The fusion technique combines the geometric depth information from Lidar with high-resolution images from cameras. This process starts with the alignment of 2D image data with 3D Lidar points using Perspective-n-Point (PnP) algorithms and Random Sample Consensus (RANSAC) to robustly estimate the pose and correspondences.

We apply semantic segmentation models to the camera images to categorize various objects within the environment. This segmentation helps in enhancing the decision-making process of the navigation system by identifying and classifying obstacles and drivable paths.
