Lidar-Camera Sensor Fusion with Semantic Segmentation

  • GitHub URL: Project Link

This project involves the fusion of Lidar and camera sensor data enhanced by semantic segmentation to improve autonomous navigation. By integrating these technologies, the project aims to create a more detailed and accurate 3D mapping of environments, which is critical for navigation in complex settings.

The fusion process combines high-resolution images from cameras with depth information from Lidar to create a comprehensive view of the surroundings. Semantic segmentation further processes these images to classify different regions of the environment, aiding in better decision-making for navigation systems.

The project employs advanced algorithms for data calibration, alignment, and fusion, providing valuable insights for applications in robotics and autonomous vehicles.
